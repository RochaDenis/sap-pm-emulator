NOTAS ABERTAS
?
Solicitações de manutenção criadas pelo agente KARL no SAP. Uma nota é o registro formal de um problema — precisa virar Ordem de Serviço para ser executada. Dados via API_MAINTNOTIFICATION / QMELSet.
