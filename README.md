# SAP PM Emulator — AxiomGO Integration Layer

This emulator provides a lightweight REST API designed to replicate the core behavior of the SAP ECC and SAP S/4HANA Plant Maintenance (PM) modules. Built with Node.js and SQLite, it comes pre-loaded with rich, realistic industrial data representing a Brazilian petrochemical plant.

## Quick Start
```bash
npm install
npm run seed
npm run seed:s4
npm run dev
```
